# Shared example components

Reusable React UI components for the `deck.gl-raster` examples — built on
[Chakra UI](https://chakra-ui.com/) v3. Internal to this repo; not published.

Consumed as a pnpm workspace package: examples list
`"deck.gl-raster-examples-shared": "workspace:*"` and import from the package
root, e.g. `import { ControlPanel, DeckGlOverlay } from "deck.gl-raster-examples-shared"`.
