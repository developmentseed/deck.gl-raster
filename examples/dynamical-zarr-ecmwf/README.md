# Dynamical Zarr ECMWF

## Setup

1. Install dependencies from the repository root:
   ```bash
   pnpm install
   ```

2. Build the packages:
   ```bash
   pnpm build
   ```

3. Run the development server:
   ```bash
   cd examples/dynamical-zarr-ecmwf
   pnpm dev
   ```

4. Open your browser to http://localhost:3000
