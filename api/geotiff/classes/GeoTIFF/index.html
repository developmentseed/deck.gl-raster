Class: GeoTIFF

Defined in: geotiff.ts:28

A high-level GeoTIFF abstraction built on @cogeotiff/core.

Separates data IFDs from mask IFDs, pairs them by resolution level, and exposes sorted overviews. Mirrors the Python async-geotiff API.

Construct via GeoTIFF.open(source) or GeoTIFF.fromTiff(tiff).

fromTiff()​

static fromTiff(tiff, dataSource): Promise<GeoTIFF>

Defined in: geotiff.ts:118

Create a GeoTIFF from an already-initialised Tiff instance.

All IFDs are walked; mask IFDs are matched to data IFDs by matching (width, height). Overviews are sorted from finest to coarsest resolution.
