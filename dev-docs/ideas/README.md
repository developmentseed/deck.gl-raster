# Ideas

Future plans that don't have an implementation yet.

This folder collects design proposals we've thought about but deferred — refactors, architectural changes, or features whose rationale is worth preserving but whose implementation isn't on the immediate roadmap. Anything here is fair game to promote into [`../specs/`](../specs/) when its time comes; until then, read-only context for future maintainers.

Distinct from:
- [`../specs/`](../specs/) — committed designs ready for or backing implementation work.
- [`../plans/`](../plans/) — gitignored, ephemeral implementation plans tracking active work.
